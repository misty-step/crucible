EVAL AND BENCHMARK WORKBENCH

Measurement discipline for AI systems.

Crucible designs, runs, judges, calibrates, and reports evals for Misty Step's AI and agent work. Every rate carries a confidence interval. Every judge needs calibration before it is trusted. Rank gaps inside the noise floor come back inconclusive, not as wins.

CLI, MCP server, and workbench UI read the same run ledger.

Why it matters

author A validated EvalSpec. measure A Wilson interval on every rate.
compare Admits the noise floor. judge Canary-guarded, calibration-scored judges.
verify Doctor check, zero spend. interop Cerberus, Harbor, and MCP adapters.
