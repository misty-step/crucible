Release notes

Crucible does not yet cut tagged releases or publish a formal CHANGELOG — it ships continuously to master. These notes are an honest stub summarizing the most recent user-facing commits, not a versioned release train. Read the full history on GitHub.

2026-07-04 - unreleased, main branch

Run inspection and external eval import

  • Added a Trace layer so a stored run's execution can be inspected step by step, not just its final verdict (crucible-030).
  • Added crucible import promptfoo, the first external eval adapter — bring an existing promptfoo config in as a declared EvalSpec instead of hand-writing one.

2026-07-04 - unreleased, main branch

Harder judge calibration, longitudinal comparisons

  • Hardened agentic-judge calibration against known gaps in the judge-vs-human agreement report.
  • Added longitudinal score queries: compare harness and tool-allowlist configuration identity across run history, not just one-off pairs.

2026-07-04 - unreleased, main branch

Verified-live onboarding

  • Added crucible doctor: one command that proves the CLI, MCP server, serve UI, and SQLite run ledger are actually live, with no network call and no spent model credit.
  • Closed a brand gap so the flask-conical mark renders consistently across the dashboard and the adjudication panel.

Evidence: cargo run -p crucible -- doctor --json exits 0 with every check ok.
